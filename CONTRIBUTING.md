# Contributing to Bilateral Stimulation Lightbar

First off, thank you for considering contributing to this project! It's people like you that make open source such a great community.

## ü§ù How to Contribute

### Reporting Bugs
If you find a bug, please create a new **Issue** describing:
1. What went wrong.
2. Steps to reproduce the error.
3. What browser/device you are using.

### Suggesting Enhancements
Have an idea to make the tool better?
- Open a **Discussion** (if enabled) or an **Issue** with the "enhancement" label.
- Explain why this feature would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Make sure your code follows the existing style (clean HTML/JS/CSS).
3. Test your changes locally.
4. Issue that Pull Request!
